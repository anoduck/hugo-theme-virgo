> 为什么要把使用指南单独列出，而不是放在 README 中？

一方面，内容更新频繁（虽然没有必要），把更改放在 `README` 中需要每次都提交主题更新，然而有时候只是单纯的更新 `README` 而已。另一方面，国内网络推送到 Github 仓库的时候经常抽风，速度也慢。
